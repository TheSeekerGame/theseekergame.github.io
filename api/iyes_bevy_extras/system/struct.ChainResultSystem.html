pub struct ChainResultSystem<SystemIn, SystemOk, SystemErr> { /* private fields */ }
Expand description

Similar to Bevy’s PipedSystem, but with diverging paths for Ok/Err Results

The main system will run, returning a Result, and then one of two other systems will be run, depending on whether the result was Ok or Err. The data returned in the respective enum variant will be piped as an input.

The output value from this system is to be returned from either of the two secondary systems, whichever runs.
