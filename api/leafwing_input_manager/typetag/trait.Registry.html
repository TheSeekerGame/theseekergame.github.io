Expand description

Registry mapping unique identifiers of types to their deserialize implementations, enabling deserialization of a specific trait object type.

Register deserialize_fn as the deserialize function for id.

This method is infallible, but errors such as multiple registrations for id may be propagated to deserialization-time by making get_deserialize_fn return an error.

Â§Errors

Implementations may return the following errors:

  • GetError::NotRegistered { id } if no deserialize function was registered for id.
  • GetError::MultipleRegistrations { id } if multiple deserialize functions were registered for id.
