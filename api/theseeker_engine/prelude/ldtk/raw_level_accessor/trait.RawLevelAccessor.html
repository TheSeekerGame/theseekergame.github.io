Expand description

Convenience methods for accessing raw level data by reference.

§Root vs world levels

This trait is intended for types that store LdtkJson data. This sort of data stores levels both in the “root” of the project, and in each World. The former is referred to in trait methods as root_levels, and the latter as world_levels. Root levels may be removed in the future after LDtk’s multi-worlds update.
