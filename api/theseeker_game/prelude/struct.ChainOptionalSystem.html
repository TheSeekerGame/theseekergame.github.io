    system_in: SystemIn,

Expand description

Similar to Bevy’s PipedSystem, but the second system is optional

The main system will run, returning an Option, and then, if the value is Some, the secondary system will run. The value will be piped in as an input.

The output value of this system is produced by the secondary system, or the default value if the primary system returns None.
