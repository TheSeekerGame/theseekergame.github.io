Expand description

Provides methods for managing the available “console commands”

There are two kinds of commands:

  • “args” (system takes Vec<String> as input)
  • “noargs” (system takes no input)

When you try to run a cli string, the correct variant will be chosen based on whether args were present.

You should be able to do this at App creation, as well as later at runtime, with exclusive World access.
